                                                                                              事件整体经历 #

                                                                                              以下是依据《平糧實蹟》原文内容，按时间顺序整理的完整事件过程：

                                                                                              1. 明初至正德年间（1368-1512） #

                                                                                              • 洪武七年（1374）：信国公（汤和）奉命履亩定税，因浦江政内乡地处山区、土瘠租微，税粮独轻，七乡科则差异确立。
                                                                                              • 建文庚辰（1400）：浦江廸功郎刘彦飞试图统一七乡科则，邑民毛伯玉上书毛尚书，以“土产肥瘠不同”为由反对，事遂寝。
                                                                                              • 正德七年（1512）：知县邹敏行清查田粮，沿丘履亩丈量，仍按旧科则造册。

                                                                                              2. 嘉靖三十年（1551） #

                                                                                              • 二月：知县韩叔阳重新丈量田亩，拟将新升官米均减，命都长赴司呈请。
                                                                                              • 十一月十日：里老金朝兴等告“清查田粮以革奸弊”，获布政司、分守道、分巡道批示，要求金华府勘处。

                                                                                              3. 嘉靖三十一年（1552） #

                                                                                              • 四月三日：奉户部札例，要求均平田粮，但强调“原额不亏、不便均者不必丈量”。
                                                                                              • 四月二十九日：韩叔阳发告示，宣布七乡科则通融均为一则。
                                                                                              • 五月三日：政内乡石文济等上告，反对均则，强调山区田瘠，若均则山民将代平原纳重税。府批“照旧例回报”。
                                                                                              • 五月四日：灵泉乡朱子高等上告，支持均则，称旧制导致富户隐田、贫民赔粮。
                                                                                              • 五月十九日：藩司批朱子高状，要求府查议。
                                                                                              • 五月二十二日：粮里老张元等再呈，巡按御史林批“查究议处”。
                                                                                              • 六月：西市老农楼永富进县请愿，韩叔阳拒之。
                                                                                              • 七月十三日：石文济等再告灵泉乡豪右贿吏变乱科则，御史林批府查报。
                                                                                              • 七月十六日：浦江县申文，称已丈量完毕，七乡结状“允服均粮”，但实际政内乡被强逼结状。
                                                                                              • 七月二十八日：浦江县申诬陈贤四十七等“捏状扰官”，分守道批“候详允”。
                                                                                              • 九月六日：陈仕正告韩叔阳贪赃，指控其受贿改科则。按察司准提审。
                                                                                              • 九月十四日：黄宗鲁等告韩叔阳贤能，布政司批“查勘回报”。
                                                                                              • 十月十七日：按察司押解韩叔阳等至杭州、衢州会审。
                                                                                              • 十月二十日：陈华告吏书变乱旧制，均粮致山民困苦。会审后韩叔阳复职，陈仕正发监追赃。
                                                                                              • 十一月三日：察院命义乌县丞周协同浦江县踏勘，造册上报。

                                                                                              4. 嘉靖三十二年（1553） #

                                                                                              • 二月七日：义乌县丞周踏勘后申文，确认政内乡田瘠，灵泉乡田肥，建议“照旧制，政内免半里差役，灵泉减三分之一”。
                                                                                              • 二月十五日：主簿杨楫申文，请求按草册造黄册，府批“五日内速完”。
                                                                                              • 四月二十九日：陈贵等四十五人告“冤狱未疏”，府批东阳、永康知县会勘。
                                                                                              • 五月八日：生员陈昌言上书知府洪公谐，详述山乡十害，反对均粮。
                                                                                              • 六月二十七日：洪公谐行牌，命“原清切者不必丈量”。
                                                                                              • 八月十八日：蒋康九告“变乱版籍”，御史赵批府推官问详。
                                                                                              • 九月二十三日：陈子镐等再告，御史赵、王批分守道查报。
                                                                                              • 十月四日：分守道右参政吴批“查勘有无买嘱作弊”。

                                                                                              5. 嘉靖三十三年（1554） #

                                                                                              • 八月九日：朱八四（由陈子镐抱奏）赴京奏疏，控灵泉乡豪右飞粮害民，要求恢复洪武旧制。九月二十六日准奏，命浙江巡按查办。

                                                                                              6. 嘉靖三十四年至三十七年（1555-1558） #

                                                                                              • 三十四年六月：按察司提审朱八四、陈贤四十七等百余人，案件迁延。
                                                                                              • 三十五年六月：浦江县许宗谥申文，建议“政内乡减米80石，灵泉乡增80石”。
                                                                                              • 三十六年九月：朱八四等再告，府批“查处舆情”。
                                                                                              • 十二月十五日：吴孟用等结状，七乡愿照旧输纳。
                                                                                              • 三十七年二月：金華府审议，认为“七乡旧额已定，不必均平”。
                                                                                              • 八月三日：巡按御史王、分守道翁等会审结案，维持旧制，事始寝。

                                                                                              7. 万历年（1586） #

                                                                                              • 万历丙戌（1586）：灵泉乡总书黄克辉篡改科则为“照亩派征”，陈九经（陈昌言后人）控诉。府县勒碑定十六则，黄克辉拟充军。

                                                                                              8. 康熙年（1689） #

                                                                                              • 康熙己巳（1689）：灵泉乡陈应祚等再倡“官民田均则”，政内乡九十六人联名反对，县府核查后维持“照土定则”。

                                                                                              序言 #

                                                                                              平糧實蹟序

                                                                                              [Page 36] 浦江定例一十九則，吾鄉稅畆獨輕。大明至今三百餘年薄賦，相因不改，論者竊云：泉寒土瘠，則壞（壤）1之例宜然。顧安享成功，伊誰之力，至此葢澤及一鄉，一齋公之勲名至今膾炙人口。而訐訟連年，諸同事之俠烈，當時艱苦備嘗。

                                                                                              按洪武七年命，信國公履畆定額。因本鄉僻處山隅，土薄租微，稅糧獨輕。

                                                                                              嘉靖壬子，大造賦役王册，當事悮信靈泉鄉豪右，請加政內鄉額米七百餘石。府縣舉行新式變法，加徭牢不可破矣。

                                                                                              旹有連名具呈者石文濟、陳贒四十七等，憫鄉人之塗炭，首倡義舉。然歷盡諸艱，曾無挽回之益。遷延數載，反獲抗官之條。甚至周諱益兒字玉三者，竟為豪右所傷，府縣俱置之不問，草菅民命抑又甚焉。

                                                                                              勢不得已而覆鼓敝兵，更堅不仆者，仕進公與石、朱、蔣、虞諸公，相繼訐控。奈士民心懷金石，而府縣已案定如山，五六年來苛征不改，七八十人羅網盡罹，脛腿二㫁，三木囊頭矣。雖停獄緩刑，賴有陳子鎬之抱奏，而給半減半之徭，畸重畸輕之罪，究未能誠服，均平[Page 37]脫然無累也。

                                                                                              幸光明石公奮其憤懣之氣，劈開畫地之囚。而一齋公乃能潛水匿逃，赴闕奏疏，困頓窮途，始得命下。及八四朱公聞疏創起，獄內申呈因，而李府主諱一元者執奏，矢公遵旨頒行炤舊，通詳各憲定則，移賚戶部，用是囚無冤獄，巷有衢歌，士民無恙，薄賦依然。

                                                                                              獨怪2當日諸公非膏梁子弟，即壯志書生，乃至捐棄身家，不避險患，禁獄囚牢。猶幸一息尚存，流杖軍徒咸受，無辜荼毒，又□怪一齋公以經濟材，請纓獲罪，坎坷終生。3

                                                                                              追念及斯，誰不為之唏噓泣下。嘗讀昌言公上府主洪公書，痛悉山鄉之苦、供億之艱。向非有人焉，協力任事於前，則當日之困於摧科，而今日之疲於給公者，不知何如也。向非立不可㧞之案於前，則萬歷丙戌之假借經濟，而康熙己巳之引例扒平者，其猖獗又不知何如也。

                                                                                              唐之平蔡也，破蔡者李愬耳。而淮西一碑，不專歸功於愬者，誠以十六道之戰勝攻取與有力焉。由今擬之，周益兒等之死於豪右也，其武元𢖍之被刺與；石文濟等之倡始也，其顏允武之合攻與；仕進公等之再[Page 38]行義舉也，其裴度之復起視師，誓必滅賊與；石光明之奮不顧身也，其董昌齡之順死賢於逆生與；朱八四之百折不磨，疏後申呈，令各院有不得不遵之勢也，其李祐之入蔡以擒元濟與；至若一齋公，乃始敗於朗山，而成功於雪夜者與。盖論，其報復之難，較之卧薪嘗膽吳市吹簫者為更甚，而究其功之所及，即免衢婺丁銀、薄青田稅畆者，無以過焉。

                                                                                              績奉勘合內開繁昌、桐陵、高諒等縣，為因科則不均，欺隱田粮，俱各具情奏准，差官丈量，清徹欺隱，但見存田地不分高下，均為一則。是以奸民無所庸其弊，愚民不致受其辜，徵納兩便，獘根可除，天下通行，萬民稱快，切念本縣原立七鄉，官民科則共計八十餘等，名色多端茫昧，以至攅書弄弊，富室欺隱，虧欠額米三百餘石遺陷，通縣賠納較獘甚於繁昌等縣。正德七年，已蒙鄒老爺設法清理僅十餘年，舊弊復作，漸復虧欠如故，見蒙督令新舊監、都、保長沿坵挨踏，履畆丈量，原額雖蒙查復，是特用功於枝葉，而未蒙拔本塞源也，揆其所由，定其科則，繁昌止蒙并為一則，以杜病根，適幸前本頒行海內，機出曠感，欲將官民田[Page 47]土通融定為一則，揭榜諭眾，莫不忻感，獨有政內一鄉，原則果輕，獲利已久，結黨阻滯，深圖自便，切彼鄉當開國之初亂離之後，地僻人稀，意在撫集，所以定稅獨輕，即今民聚地闢，風氣大開，實為財藪，與各鄉利甚懸絕，實迹昭彰，自不容掩，設拘於舊例則彼世受其利，而各鄉咸受其害，况俗有推遷，法有變通，安得以一鄉之私情，而梗朝廷之良法，若不急上告，鳴失此機，會更無伸洩，切如嘉靖二十年間，蒙監察御史奏為清乞田賦以成盛世事例，准行府縣，又被伊鄉夤緣不行，似此得計，仍圖僥倖，伏懇天心遵照事例，即將官民田地均為一則，雖使歸女稚子亦知所以筭計辦納，而老奸巨猾無所措手用計，一勞永逸，造福無窮，有此利害，具狀上告。

                                                                                              嘉靖三十二年（1553）5月8日，陈昌言，上书洪公谐为恳乞恩庇广开言路以固民瘼以救生灵事 #

                                                                                              金華府浦江縣儒學治下增廣生員陳昌言，係政內鄉二十六都一啚民籍，謹熏沐頓首百拜肅書進呈，

                                                                                              即師相大柱國太祖父母老大人先生䑓下：

                                                                                              書為懇乞恩庇、廣開言路、以恤[Page 61]民瘼、以救生靈事。

                                                                                              念言不幸生居山谷，忝蒙作養，自守頗堅，日誦聖賢之書，時凜卧碑之訓，每期致知允踏，以求無負聖諭、古人教戒也，並不敢輕於干謁，以自喪所守，但有關於一鄉之利病，在山民既限於不能言，在案詞又病於不能詳，在言又謹守，夫惟生員不許之戒，是以下情終無由以上達，而民力之殫、民膏之竭、亂心之起，職此故矣。言今事幹切身，甘冒罪貫，敢昧死為大人言之，伏乞廣開包荒虛心體物宥罪釆納，則大人再與山民造命而言之，所學為無負矣。

                                                                                              切念浦江地有山谷平陽肥瘠高下向背寒暖之別也，故禹貢曰：“田有三等，賦有九則”，詩曰：“相其陰陽，觀其流泉”。是以我太祖高皇帝龍飛淮甸、迅掃腥羶、肇造寰宇、委改定則，靡不竭盡心思，勞費謀畫。

                                                                                              因浦江土有七郷之異，而定為七則，良有以也。去年韓父母欲將通縣均為一則，以革欺隱之弊，甚盛心也，但五鄉之米俱得如故，均得固可，不均亦可，是以緘黙不言。獨政內鄉之山田，舊額米不過七百余石，而新加之米反倍加一千五百餘石，至於靈泉鄉之米，減去六[Page 62]百餘石，是山民代靈泉人納稅當差，而彼享安逸之利也。此政內鄉獨於靈泉牴牾，而迄今事之未成也。近承上司明文“已照舊造冊解京司矣，府縣冊已造完矣，夫何今日又有平米之事乎？”其費過紙張，勞盡民力。不暇殫述言且備以政內之不如靈泉者，先敬陳之，而後及于平米之利害，試垂聽焉。

                                                                                              彼七郷之米政內惟輕，而靈泉稍重者，豈聖祖委官定則之初厚薄於其間哉，由政內之害有十，而靈泉之損無一，故也。

                                                                                              是姑夾山兜水叠石為田，而廣平坦夷者不靈泉也；山高水冷五穀不結，而暖氣長苗者不靈泉也；鳥鼠交侵野猪猿食，而日夜不守者不靈泉也；流沙淤塞草木蔭蔽，而避水開陽者不靈泉也。

                                                                                              狼虎傷噬則鳴金持械以驅逐，其能若靈泉之清靜安逸乎？水田一畆則收穫止有二石，其能若靈泉之高田而反倍得者乎？山穀𢡱厚則舂米已折其半，其能若靈泉之斗穀而有八升者乎？

                                                                                              吾見靈泉塘池廣闊，而山間之坑水易涸車戽無望者，信不如矣；靈泉土厚水深，而山間之沙田疏漏即注即幹者，信不如矣；靈泉平易便利[Page 63]，而山間之峻嶺重越商人少至者，信不如矣。

                                                                                              且山間之民生於山而不能食於山也，以歲歲仰給於靈泉者十分有九。搬運之勞則險越重嶺，洗鍋待糧則家有饑色。即遇小旱，則山間男女為靈泉之奴僕者，不知其幾也，衣服器血之當沒於靈泉者，不知其幾也。小民無資當典則惟采草根剝樹皮以救殘喘，張目待哺終日而不得一食者有之，二日始得一食者亦有之，民皆菜色而餓殍相繼，此言之目擊心痛者也，使有家國之責者，聞之不能不愴然而憫矣。

                                                                                              言生長于此，習知民病，並未見挑穀出靈泉濟貧者也，况靈泉之鄉，泉甜土肥民殷家富，栚木之植一望無際，田荳蕎麥之得廣種兼收，夏荳木棉之利多栽竝蓄。

                                                                                              在山間之民非安於逸而不種也，但山氣凝寒霜露先墜，雖種而無收也。昔人有憫農之詩雲：“二月賣新絲，五月糶新榖，醫得眼前瘡，剜卻心頭肉”。此言之親切有味無甚於山間之民矣。

                                                                                              盖山間取靈泉之當，幷償靈泉之債，不能盡皆銀穀，是故或以苦楮、或以山栗、或以靛青之類悉輸積穀之家，如一失信則荒年生路[Page 64]無望矣。是山間之所産，靈泉盡有也。彼以一穀之多而易我所盡有，亦可痛也。

                                                                                              及至秋風乍起，山內先寒，臘月之夜，則燒柴火以待天明者，吾見亦多；盖草薦以代被者，吾見亦多也。

                                                                                              夫以靈泉之利既如彼，而政內之害又如此。而猶又為均米之說者，夫豈無所據見也哉。盖其計有三，而言之易入者有五。曰遵奉勘合也，曰均米者愛民一體之心也，曰均得成韓父母永血食於浦邑也，此其計之善者也。

                                                                                              言也，雖處山谷亦嘗與聞勘合之行，為“有地者無糧、有糧者無地”，為“均地均糧”，非以槩均天下之科則也。其中有云：“不便於民者，不必一概騷擾”，是葢聖天子惠養元元之盛心，固欲為司牧者以善法其意也。

                                                                                              况七鄉之額行之已久，吏習民安。在今日順而治之可也，行其所無事可也。設使可均則，定額之初固非無人，而奚待於今日耶。必欲均之，則山民愚蠢，雖教化之則難興禮義，激之則易與為亂，亦風聲氣習然也。倘或變生肘腋，則政內必與靈泉相攻，他日養成亂階，則于韓父母之盛名有玷，而于大人之心力亦勞矣。如之何使[Page 65]愛民之心反為變民之舉，而血食之念變成召亂之媒乎。

                                                                                              何以謂其言之易入者有五？曰：山民之稠過往昔也，土地之闢過往昔也，山內有杉木生漆之財也，苧蔴桐油之利也，山禽麂皮之得也。

                                                                                              要之，杉木之植固可得利，然而萬畆荒山之中，種得一畝已是有力人家矣。雖不種亦輸稅者不知幾萬頃也。自其成材得利而言之，則杉木求用於三十年之後，苧麻油漆之類求用六七年之後，自此以前固不能年年而得利也。便是麂皮之得，出於遊手好閒之徒，然十人共逐一麂，大者不過值銀三錢足矣，使十人均分，不知可以償十人之工食否乎？況終日得一兔者有之，終日上下山林而不一得者亦有之。盖民之力氣只有此數，不在此則在彼。無田可種，故種山以度日，逐麂以求食，亦大不得已之計也。雖是山民之眾不比徃昔然，然生一人則冊增一口而宣力於國家矣。

                                                                                              言以為人固可增苐，恐封域有限而世界不可擴也。以有限之田，而供此生生不不已之人，尚慮其田畞之少，而無土營生也。

                                                                                              大人不察而信易入之言，竊謂大人民情土俗，未能[Page 66]易知也。今蒙選委賢能均為上中下三則，言恐雖不比初時倍加之重，然於政內未必不加，於靈泉未必不减也。相沿陋習時難頓革，民心思亂又乘時起矣。是於國家無益，于靈泉有利，於山民有害也。

                                                                                              况政內之米照舊當差，尚且十室九空，至於糧長一當則又室如懸罄矣。故人之報私恨者，惟願其當糧長以破家。若是者夫，豈民之頑梗而糧之難收哉。亦以一年之所收不足以供一年之所費，而錢糧之難辦如此也，以故情願賣田以避此役。

                                                                                              在靈泉之人則又百計以謀糧長之當矣。果如額外加米，則靈泉之當者益富，而山民之貧者愈貧也，斯民烏得而不具兵耶。是非好訟而好兵也。憂其里長徭役之難當，糧稅之難辦也。數年之後，官有督責之威，吏有催迫之苦。雖有好田賤賣於人，而亦無人來買矣，終至於賣妻子以完官，以及逃亡之禍者，此必然之勢也。

                                                                                              大人繹而思之，無怪山民之憂惶無措而至於亂也。設將政內之米減入靈泉，則靈泉人又與政內相爭矣。惡民之名不在政內，而在靈泉也。

                                                                                              言雖不通時務，常讀宋史，恒以為靖康[Page 67]之禍不在徽欽北狩之時，而在王安石見用之日。自今言之，安石新法之行，本欲強國而非欲誤國也，但其所學之偏，執議之堅，恐人阻格，則異同為用舍，故引結兇邪，牢不可破，而國隨以亡。千載之下尤為之太息也。大人廊廟寵才，贊襄在即，肯使安石覆轍復蹈於今日乎？

                                                                                              古人謂，處江湖之遠則憂其君，居廟堂之上則憂其民。伏乞大人以憂君之心先於憂民，絕口不言均米之事，賜山民以更生之路，則洪恩之旁，及祝頌之洽浹，合遠近而無間矣。

                                                                                              必不得已，請於通縣額外更加政內之米，或百石或百五十石，以輸於國家。則於靈泉固不減，而山民之亂心亦息矣。此最今日之急務也。

                                                                                              《書》曰：“消患於未萌，保邦于未危”。又曰：“怨豈在明，不見是圖”。《易》曰：“履霜，堅冰至。君子以防微杜漸”。敬以是為大人獻言也。目見山民蜂屯蟻聚人人思亂，萬或適遭其變，則言之妻子亦不保矣。天日之誠，誓不為習俗所移，以累聖明之治，及大人作養之恩也。行將自經林木舉家焚燎也。狂言發於忠悃，不識忌諱，實甘分罪。倘蒙俯鑒下情、虛心聽納，則生民幸[Page 68]甚、生靈幸甚。臨書不勝戰慄恐懼之至。

                                                                                              {陈昌言书}

                                                                                              金華府浦江縣儒學治下增廣生員陳昌言，係政內鄉二十六都一啚民籍，謹熏沐頓首百拜肅書進呈，即師相大柱國太祖父母老大人先生䑓下：

                                                                                              書為懇乞恩庇、廣開言路、以恤[Page 61]民瘼、以救生靈事。念言不幸生居山谷，忝蒙作養，自守頗堅，日誦聖賢之書，時凜卧碑之訓，每期致知允踏，以求無負聖諭、古人教戒也，並不敢輕於干謁，以自喪所守，但有關於一鄉之利病，在山民既限於不能言，在案詞又病於不能詳，在言又謹守，夫惟生員不許之戒，是以下情終無由以上達，而民力之殫、民膏之竭、亂心之起，職此故矣。言今事幹切身，甘冒罪貫，敢昧死為大人言之，伏乞廣開包荒虛心體物宥罪釆納，則大人再與山民造命而言之，所學為無負矣。

                                                                                              切念浦江地有山谷平陽肥瘠高下向背寒暖之別也，故禹貢曰：“田有三等，賦有九則”，詩曰：“相其陰陽，觀其流泉”。是以我太祖高皇帝龍飛淮甸、迅掃腥羶、肇造寰宇、委改定則，靡不竭盡心思，勞費謀畫。

                                                                                              因浦江土有七郷之異，而定為七則，良有以也。去年韓父母欲將通縣均為一則，以革欺隱之弊，甚盛心也，但五鄉之米俱得如故，均得固可，不均亦可，是以緘黙不言。獨政內鄉之山田，舊額米不過七百余石，而新加之米反倍加一千五百餘石，至於靈泉鄉之米，減去六[Page 62]百餘石，是山民代靈泉人納稅當差，而彼享安逸之利也。此政內鄉獨於靈泉牴牾，而迄今事之未成也。近承上司明文“已照舊造冊解京司矣，府縣冊已造完矣，夫何今日又有平米之事乎？”其費過紙張，勞盡民力。不暇殫述言且備以政內之不如靈泉者，先敬陳之，而後及于平米之利害，試垂聽焉。

                                                                                              彼七郷之米政內惟輕，而靈泉稍重者，豈聖祖委官定則之初厚薄於其間哉，由政內之害有十，而靈泉之損無一，故也。

                                                                                              是姑夾山兜水叠石為田，而廣平坦夷者不靈泉也；山高水冷五穀不結，而暖氣長苗者不靈泉也；鳥鼠交侵野猪猿食，而日夜不守者不靈泉也；流沙淤塞草木蔭蔽，而避水開陽者不靈泉也。

                                                                                              狼虎傷噬則鳴金持械以驅逐，其能若靈泉之清靜安逸乎？水田一畆則收穫止有二石，其能若靈泉之高田而反倍得者乎？山穀𢡱厚則舂米已折其半，其能若靈泉之斗穀而有八升者乎？

                                                                                              吾見靈泉塘池廣闊，而山間之坑水易涸車戽無望者，信不如矣；靈泉土厚水深，而山間之沙田疏漏即注即幹者，信不如矣；靈泉平易便利[Page 63]，而山間之峻嶺重越商人少至者，信不如矣。

                                                                                              且山間之民生於山而不能食於山也，以歲歲仰給於靈泉者十分有九。搬運之勞則險越重嶺，洗鍋待糧則家有饑色。即遇小旱，則山間男女為靈泉之奴僕者，不知其幾也，衣服器血之當沒於靈泉者，不知其幾也。小民無資當典則惟采草根剝樹皮以救殘喘，張目待哺終日而不得一食者有之，二日始得一食者亦有之，民皆菜色而餓殍相繼，此言之目擊心痛者也，使有家國之責者，聞之不能不愴然而憫矣。 言生長于此，習知民病，並未見挑穀出靈泉濟貧者也，况靈泉之鄉，泉甜土肥民殷家富，栚木之植一望無際，田荳蕎麥之得廣種兼收，夏荳木棉之利多栽竝蓄。

                                                                                              在山間之民非安於逸而不種也，但山氣凝寒霜露先墜，雖種而無收也。昔人有憫農之詩雲：“二月賣新絲，五月糶新榖，醫得眼前瘡，剜卻心頭肉”。此言之親切有味無甚於山間之民矣。

                                                                                              盖山間取靈泉之當，幷償靈泉之債，不能盡皆銀穀，是故或以苦楮、或以山栗、或以靛青之類悉輸積穀之家，如一失信則荒年生路[Page 64]無望矣。是山間之所産，靈泉盡有也。彼以一穀之多而易我所盡有，亦可痛也。

                                                                                              及至秋風乍起，山內先寒，臘月之夜，則燒柴火以待天明者，吾見亦多；盖草薦以代被者，吾見亦多也。

                                                                                              夫以靈泉之利既如彼，而政內之害又如此。而猶又為均米之說者，夫豈無所據見也哉。盖其計有三，而言之易入者有五。曰遵奉勘合也，曰均米者愛民一體之心也，曰均得成韓父母永血食於浦邑也，此其計之善者也。

                                                                                              言也，雖處山谷亦嘗與聞勘合之行，為“有地者無糧、有糧者無地”，為“均地均糧”，非以槩均天下之科則也。其中有云：“不便於民者，不必一概騷擾，是葢聖天子惠養元元之盛心，固欲為司牧者以善法其意也。

                                                                                              况七鄉之額行之已久，吏習民安。在今日順而治之可也，行其所無事可也。設使可均則，定額之初固非無人，而奚待於今日耶。必欲均之，則山民愚蠢，雖教化之則難興禮義，激之則易與為亂，亦風聲氣習然也。倘或變生肘腋，則政內必與靈泉相攻，他日養成亂階，則于韓父母之盛名有玷，而于大人之心力亦勞矣。如之何使[Page 65]愛民之心反為變民之舉，而血食之念變成召亂之媒乎。

                                                                                              何以謂其言之易入者有五？曰：山民之稠過往昔也，土地之闢過往昔也，山內有杉木生漆之財也，苧蔴桐油之利也，山禽麂皮之得也。

                                                                                              要之，杉木之植固可得利，然而萬畆荒山之中，種得一畝已是有力人家矣。雖不種亦輸稅者不知幾萬頃也。自其成材得利而言之，則杉木求用於三十年之後，苧麻油漆之類求用六七年之後，自此以前固不能年年而得利也。便是麂皮之得，出於遊手好閒之徒，然十人共逐一麂，大者不過值銀三錢足矣，使十人均分，不知可以償十人之工食否乎？況終日得一兔者有之，終日上下山林而不一得者亦有之。盖民之力氣只有此數，不在此則在彼。無田可種，故種山以度日，逐麂以求食，亦大不得已之計也。雖是山民之眾不比徃昔然，然生一人則冊增一口而宣力於國家矣。

                                                                                              言以為人固可增苐，恐封域有限而世界不可擴也。以有限之田，而供此生生不不已之人，尚慮其田畞之少，而無土營生也。

                                                                                              大人不察而信易入之言，竊謂大人民情土俗，未能[Page 66]易知也。今蒙選委賢能均為上中下三則，言恐雖不比初時倍加之重，然於政內未必不加，於靈泉未必不减也。相沿陋習時難頓革，民心思亂又乘時起矣。是於國家無益，于靈泉有利，於山民有害也。

                                                                                              况政內之米照舊當差，尚且十室九空，至於糧長一當則又室如懸罄矣。故人之報私恨者，惟願其當糧長以破家。若是者夫，豈民之頑梗而糧之難收哉。亦以一年之所收不足以供一年之所費，而錢糧之難辦如此也，以故情願賣田以避此役。

                                                                                              在靈泉之人則又百計以謀糧長之當矣。果如額外加米，則靈泉之當者益富，而山民之貧者愈貧也，斯民烏得而不具兵耶。是非好訟而好兵也。憂其里長徭役之難當，糧稅之難辦也。數年之後，官有督責之威，吏有催迫之苦。雖有好田賤賣於人，而亦無人來買矣，終至於賣妻子以完官，以及逃亡之禍者，此必然之勢也。

                                                                                              大人繹而思之，無怪山民之憂惶無措而至於亂也。設將政內之米減入靈泉，則靈泉人又與政內相爭矣。惡民之名不在政內，而在靈泉也。

                                                                                              言雖不通時務，常讀宋史，恒以為靖康[Page 67]之禍不在徽欽北狩之時，而在王安石見用之日。自今言之，安石新法之行，本欲強國而非欲誤國也，但其所學之偏，執議之堅，恐人阻格，則異同為用舍，故引結兇邪，牢不可破，而國隨以亡。千載之下尤為之太息也。大人廊廟寵才，贊襄在即，肯使安石覆轍復蹈於今日乎？

                                                                                              古人謂，處江湖之遠則憂其君，居廟堂之上則憂其民。伏乞大人以憂君之心先於憂民，絕口不言均米之事，賜山民以更生之路，則洪恩之旁，及祝頌之洽浹，合遠近而無間矣。

                                                                                              必不得已，請於通縣額外更加政內之米，或百石或百五十石，以輸於國家。則於靈泉固不減，而山民之亂心亦息矣。此最今日之急務也。

                                                                                              《書》曰：“消患於未萌，保邦于未危”。又曰：“怨豈在明，不見是圖”。《易》曰：“履霜，堅冰至。君子以防微杜漸”。敬以是為大人獻言也。目見山民蜂屯蟻聚人人思亂，萬或適遭其變，則言之妻子亦不保矣。天日之誠，誓不為習俗所移，以累聖明之治，及大人作養之恩也。行將自經林木舉家焚燎也。狂言發於忠悃，不識忌諱，實甘分罪。倘蒙俯鑒下情、虛心聽納，則生民幸[Page 68]甚、生靈幸甚。臨書不勝戰慄恐懼之至。

                                                                                              （民国本位于后面标注处。乾隆本无第一段小字注。）

                                                                                              嘉靖壬子，邑侯韓叔陽大造賦役黃冊，聽信靈泉鄉豪右朱文經，更改七鄉輸糧科則，重加政內鄉額米七百余石。府主洪公諧力主新法苛征，民不勝害。公念奔告水程具疏赴奏。其疏曰：

                                                                                              浙江等處承宣佈政司使金華府浦江縣儒學廩生臣陳九經，奏為奸豪蠱政亂法更制酷害蒼生事。

                                                                                              臣聞夏禹鑄九鼎以分九州，則壤成賦不以一例病民。降及商周，鄉遂都鄙，隨其地土所宜，以定公田祿養，正其經界，不致豪強之兼并，不為貪暴之多取，循古典也，何事於紛更哉？

                                                                                              商君開阡陌以反古，恣慘毒以亂秦，道途側目、閉口含冤，更法何利於國家哉？漢唐宋曆遵舊籍[Page 81]，雖加損益，不過因時制宜，期與百姓相安而已，並不紛更田制，以拂民情。間有王安石突創青苗法，圖為興利，詎知反害我蒼生。

                                                                                              我太祖高皇帝，軫念民隱，以賦稅為國家重務、錢糧乃下民膏血，關係匪輕，不許亂行更變。戶部鑄銅版以定科則、糧長，給勘合以禁奸貪，其為國計慮者至深切矣。

                                                                                              蘇松等府，原因富戶資助張士誠，負固不服，故照其租簿之額數，以定稅糧之規則，征之太重，洪武十三年加恩存恤，特命戶部為之遞減，其為民情計者，誠優渥矣。

                                                                                              浙東山郡，僻在一隅，田有水平高，地有上中下。洪武七年，命信國公延坵履畞，相土定糧，皆因循古制隨在宜民。

                                                                                              本邑靈泉鄉，附郭平陽曠原沃野，土厚租多，水田每畆科糧四升四合，比之浙西四斗科徵者為尤輕矣；政內鄉叠嶂層巒、山高水冷，其土薄而租微，水田每畞科糧一升九合，比之青田三合科徵者為尤重矣。因土之厚簿以定糧之輕重，祖宗成法萬古良規。

                                                                                              豪右鼓惑縣官積奸乘機射利。一旦更變舊制，不論地利厚薄，槩以一例取盈，每畞併科三升二合，巨屨小屨[Page 82]同價，是亂天下也，何利於生民哉？

                                                                                              赴愬者坐獄，訟獄者連刑，坑戮儒生，荼毒良善，是害百姓也，何取於新法哉？

                                                                                              民困征誅，望風逃竄，山鄉竟為廢土，額糧何所充納，征役何所征輸，其為利害何如哉。更新變舊，又非額外加增，平糧何益於國，不平何損於國，其為是非何如哉。

                                                                                              万历年平粮 #

                                                                                              [Page 86]萬歷丙戌，靈泉鄉總書舞獘，將通縣田糧一概炤畝派徽。先祖陳九經控告各憲。

                                                                                              該縣靈泉、政內鄉，七鄉田土，各自有肥磽不同，因土定則，亦各自有輕重不等。節經多官勘審據擬炤米派徵。

                                                                                              歴年成案具存。遍邑民情永協，向遵行無異議矣。乃總書黃克輝倚恃衙門積戀，輒敢舞文爲奸，計將通縣田糧槩為照畞攺派，因而搆通棍徒戴邦明等，駕空揑告罔上行私，鼓動鄉民鬨然告擾，且陽假仗義執言之舉，陰圖科歛索詐之謀，此真治世之亂民也，若不重加究治，何以肅法紀而儆將來。

                                                                                              黄克輝引例擬遣，允快輿情。戴邦明姑坐誣徒，陳石虞詞有添揑，情非得已，姑杖未到。方叔敬揑詷揑帮，應另提結。

                                                                                              其槩縣七鄕田糧俱從舊制照米派徵。

                                                                                              取：違制引例充軍：黄克輝。徒罪：戴邦明。誣告杖罪：陳石虞。不應炤出：方叔敬。另為具招。於萬歷十五年二月廿一日解。

                                                                                              蒙帶管分守道批審得：積書黃克輝變亂成規，主唆戴邦明揑詞罔上，擬遣允當其辜苐，本犯尚執以刻單爲詞。仰府再一鞠以服其心。繳。行間本年三月內。

                                                                                              蒙廵按傅按臨會審得：黃[Page 87]克輝以積書為奸,希圖誆騙串通戴邦明等告，欲更易成規，此等積戀若不速加重治，法紀安在耶？仰府三自內招解，行間本年四月內。

                                                                                              蒙恤刑游臨會審得：丈量之禍，流毒海內，止以狐鼠之輩，因緣爲姦，故其創愈深而其殃愈烈也。黃克輝乘機網利，紊亂成法，不痛懲之無以謝黔首與，戴邦明俱聽該府畫法究擬詳行。

                                                                                              蒙府嚴提方叔敬，拘取黃克輝、戴邦明等一千（干）人卷到官，細加嚴審，將虞士幸、倪戴周、樓志端等衆執。

                                                                                              萬歷十四年，分本縣一應錢糧由票，委係克輝赴道筭派，獘攺照畆發式等情。

                                                                                              看得該縣七鄉田土之肥磽，糧則之輕重，原有差等，然於七鄉之中，靈泉平明稱上壤，故畆闊而米則稍重，政内山鄉稱下壤，故畆狹而米則稍輕。此自國初以來至今，節經府縣勘擬，俱爲照米派徵，遵行日久，民且相安，世世不易之良規也。而摠書黃克輝敢倚積戀，舞文為奸，乃乘筭派錢糧，輙以照米私攺照畆，簧鼓人心，變乱成法，真為惡之首、而罪之魁矣。夫由票雖奉糧署道頒發，筭派錢糧數目皆出克輝之手，又安得以刻單而藉口也。擬例[Page 88]問遣且有餘辜。戴邦明、方叔敬乘機揑告紛然，擬例問徒。夫復何辭。

                                                                                              府主鲁如春，節推徐萬仞，縣主俞祖善，通融酌議，七鄉田土各自以爲輕重科，定十六則，鄉額無虧，府縣勒碑。

                                                                                              康熙年平粮 #

                                                                                              大清康熈已巳，浦江縣合邑士民陳光發、黃宗魯、陳應祚、張芹等，呈為籲憲廣佈覃恩，剔弊甦民，以埀不朽事。

                                                                                              政簡則民易從，緒繁則獘日積。念浦弹丸僻壤，田地山塘雖少，派徵科則較多。官田有五，民田有八。民田每畝自三分、五分以迄九分零，官田每畆自一錢二分、三分以至二錢五分止。官田民田數幾倍蓰，一應里甲差徭、人丁雜派，俱照正糧供應，凡管（采字頭）官田糧重者一至，現年里長苦累紛碎難堪，變賣無由，白送莫受，禍係釘骨，害深黑海。及大造推収之年，為獘尤甚，承管里保冊科等役，賄行欲肆，以官易民，以民作官，蚩匕之民聽其飛洒無從查考，堂匕縣令習於積獘不暇覺察。狡兎獲免，耿雉罹殃。總因事關國課，誰敢妄陳更議。

                                                                                              今幸遇聖主覃恩諭，著經管錢糧各司道詳察，有無細閱，有無應行更攺増刪，務期永[Page 89]有可行，而各憲隨體通行下頒，誠為天地立心，爲民物立命也。

                                                                                              現有永康縣與浦同患，己經詳請田地山塘官民扒平一則起科之文，蒙批在案，如浦亦得與永康并金蘭等一視同仁，則士民易以輸將，司令不難考成。且編審之期督，徵冊籍條貫不紊，申送核覧簡易無擾。伏乞憲天大老爺嚴行該縣，務將田地山塘無分官民，祇就四項悉歸一則，究追通邑錢糧，復還舊額，一體遵依，毋致揑覆，轉詳各憲，上以布聖天子之宏恩，下以甦窮民之困苦。迫切連名上呈。

                                                                                              署府主常批仰浦江縣，確查速報，四月初十日准

                                                                                              浦江縣趙示：着七鄉酌議詳覆。

                                                                                              政內鄉合鄉士民，呈爲懇仍古制恩賜詳覆懲究刁訟以安民生事。

                                                                                              切浦自唐建邑以來，隨分七鄉，不以一則科徵，因田土之肥磽定糧額之輕重，其宋元明歷遵舊制。

                                                                                              洪武七年命信國公延坵履畆，相土定額，隨在宜民。

                                                                                              至嘉靖壬子，有靈泉鄉土豪朱文經，更攺七鄉科則，鄉有廩生陳九經，具疏奏聞朝廷，命下撫按，轉行司道府主，勘問奸豪擬徒，舊制不易。

                                                                                              萬歷丙戌復有靈[Page 90]泉鄉摠書黄克輝,假京制全書爲聖旨，夤謀焰畆科徵，經復奔控各憲，准飭府縣酌議，七鄉田土各自以為重輕科，定一十六則，鄉額無虧。戶部鑄銅版以定科例，府縣勒碑石以垂不朽。

                                                                                              卽聖天子御極以來，俯察民情，剔獘防奸，於康熈三年間，頒行清丈田土，有則升科，無則汰除，並無更科改則之舉，忽今復出靈泉鄉，行險潑衿，陳應祚、張芹等，勢倚田連阡陌，賄通縣覇檮杬傅旭元，仗餉衿爲護符，希攺成規。陽假仗義執言之舉，隂圖科斂索詐之謀。霹造合邑士民公呈，假借多則多獘名色，揑陳光發、黄宗魯等鬼名，以籲憲廣佈覃恩等事，聳誑府憲，送臺查覆。

                                                                                              切靈泉鄉平陽沃野，土厚租多；政內鄉山高水冷，土薄租微。何堪概以一例取盈，設以一則科徵，不分田主厚薄，姦豪得計，良善荼毒，山鄉之民亦何樂耕於其野。今豪以官額病民為詞，切官額七鄉俱有官業，授受之際預先稽察，里書冊保何得舞獘於其間。總之，土有厚薄，糧分輕重[Page 91]，譬之温處之産,不能比例於蘇松，蘇松之糧何能均加於溫處？且古制彰彰於勘合，煌匕於縣誌，民情協而遵行乆。一旦紛更，多事其如銅版碑石何？

                                                                                              奸書舞文已經重擬，豪衿鼓政豈容寛宥。

                                                                                              隆遇青天仁政，宜民清亷，首出浙省，神明佩服。

                                                                                              遐懇恩賜詳覆仍遵古制，救民樂業，斬奸除害，甘棠不朽。

                                                                                              急恶切連名抄粘原疏上呈。

                                                                                              縣行府詳文。

                                                                                              合邑士民陳光發呈詞等情，前赴署金華府事。本府清軍總捕廳加一級常案下呈准。

                                                                                              蒙批仰捕江縣：確查速報等因到縣

                                                                                              蒙此遵卽出示傳集原呈士民查議詳間

                                                                                              據政內鄉士民九十六人，呈為懇仍古制，恩賜詳覆，懲究刁訟，以安民生事。

                                                                                              查看得浦邑田土厚薄不同，附郭平陽其土膏厚，僻壤山谷其土瘠薄。照依田土之厚薄，以定糧之輕重，經制分晰各則，歷代相沿，若遽為紛更移易，不分官民一例均輸，不惟變亂成規奸頑得計，而糧重者既以減輕爲得計，則糧輕者必以加重為苦難，如此則使之爭訟紛紜，將必致[Page 92]逃亡見告矣。

                                                                                              今據士民陳文耀等具呈前來，卑職核詳情由，理明詞正，而先朝奸民更攺告訐之案，有陳氏譜牒可稽，應遵舊例，不便紛更，今奉批查理合詳覆。為此具文申伏候憲裁定奪。
