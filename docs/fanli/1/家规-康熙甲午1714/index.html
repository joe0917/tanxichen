                                                                                              家规 康熙甲午（1714）

                                                                                              〇 家規 #

                                                                                              本文链接

                                                                                              • —祭祀務在孝敬，以盡報本之誠，其或行禮不恭、昭穆失次、與夫跛倚欠伸、噦噫1啑(嚏)咳，一切失容并備祭者，或臨時失落祭品，遅延過期者，家長議罰。

                                                                                              • —家長當以至誠待下，言不妄發、行不妄為，庶合古人修齊之意，至族中有關倫理之事，既經投祠，告愬其是非曲直，亦必公平剖處，以歸和好，不得膜外視之，摸稜(棱)偏護，致失宗族之誼。

                                                                                              • —子孫有虧倫理，重懲不宥，或因産業、或因小忿，敢揑(捏)閨閫誣投，以上污宗祖下辱子孫，此猶禾中之蝥、人中之獣也，察實歹坐，從重議罰，決不少貸。

                                                                                              • —子孫須恂恂孝友，實有義家風氣，見兄長坐必起，行必序，應對必以名於尊長，咸以正稱，不評假名易字，兄弟相呼各以其字冠於兄弟之上，叔伯之命姪亦然，稱伯叔則以行繼之，以父諸婦姊姒相呼并同。

                                                                                              • —卑幼不得抵抗尊長，其有出言不遜、制行悖戻者，姑誨之，誨之不悛，授以家法。子孫受長上訶責，不論是非，無得分理，但子孫固當竭力以奉尊長，尊長亦不可挾此自尊，攘拳奮袂，忿言穢語，使無所容，致失教養之道。

                                                                                              • —子姪雖年至六十，母(毋)得與叔伯連坐，會膳飲食必後長者，應對賓客不得雜以裏俗方言，謔浪敗度，凡諸舉動不宜掉臂跳足以陷輕儇。

                                                                                              • —子孫不得惑於邪説，創造異端，祠宇粧塑土木彩像，以滋妖妄犯之者，家長必嚴加訓諭，倘或怙過不悛，則以不孝論，生不許其同譜，死不許其入祠。

                                                                                              • —一粥一飯來處不易，半絲半縷物力維艱，古訓有之，子孫不得鬬勝炫奇，有傷湻樸之美，彼以其奢我以吾儉又何害哉。

                                                                                              • —棋秤、雙陸、詞曲、蟲島、獵犬之類，皆足以蠱惑心志，廢事敗家，子孫當一切弃絕之。

                                                                                              • —子孫不得從事交結，以促助閶裏為名，恣行已意，遂至輕冒刑憲，墮圯家業，切宜警戒。

                                                                                              • —子孫賭博無賴許，親房會家長重罰，仍踏不悛，則授家法，又不改則在公而放絕之。

                                                                                              • —子孫毋習吏胥，毋為僧道，毋狎屠豎，以壞亂心術，玷污祖宗，當時以仁義兩字銘心鏤骨。

                                                                                              • —為學須以孝義時務，若一向偏滞詞章深所不取，故訓誨子姪必須延迎端方禮法之士，庶子弟有所觀感、有所興起，其於學問亦資益非小，若嚨詞幻學之流，當巽詞以謝之可也。

                                                                                              • —婚嫁必須擇家世湻良者，不可苟慕富貴，以貽蔓附之譏，虧擇配之義，共豪強逆亂、卑污悪疾，諸家毋輕與義。

                                                                                              • —世人生女往往多致淹沒，縱曰女子難嫁，裙布荆釵亦何不可，違者重罰不貸。

                                                                                              • —女子年長者不許隨母出外，雖至親之家亦不許往，違則重罰其父母驕縱之過。

                                                                                              • —庶母祇受自己子婦拜跪，其餘子弟不過長揖，諸婦並同，死後忌日並同，而平時晉接，則雖嫡子嫡婦，亦須存先一飯之禮，其或侮傲不恭，家長議罰。

                                                                                              • —男女授受不親，禮也。總貧窘不得假令妻女往來貸借，與鄰裏交談接笑，以失體統，違則兩家同責。

                                                                                              • —臨喪當務盡禮，不得惑於釋老陰陽、非禮拘忌之說，以乖大義有志者，其儀式當遵文公家禮。

                                                                                              • —壯婦類多無識之人，最能翻鬬是非，若匪高明貞淑，鮮不遭其聾瞽，切不可縱其往來，歲時慶賀亦不註私入房闥，違者重責。

                                                                                              • —増拓産業，彼則出於不得已，吾則欲為子孫悠久之計，當體究果值幾緡，公平交易切勿與駔獪交謀，潛萌侵人利己之心，否則天道好還，縱得之必失之矣，更有肩挑小賣，為利甚微，更不可倚勢欺凌，自傷忠厚，違者重罰。

                                                                                              • —子孫讀書務須實學，不得寓目淫書，旁及符呪(咒)，習成匪類，有乖正業，至於會文講學，正當切切偲偲，毋尚非剌，致傷雅誼。

                                                                                              • —子孫儻(倘)有出仕者，當以報國愛民心，如有欺君虛民，生則削譜，死則不准入祠，果被誣指，不拘此例。

                                                                                              • —宗人實同一氣，切不可恃勢凌鑠，以忝厥祖，有無相通，患難相恤，庶成雍陸。

                                                                                              • —凡人子生日，此父母劬勞之日，如父母具慶，當置酒以奉歡娛，父母俱亡，當倍加悲痛感恩報本之道也。再祖父陰壽，做戲燕客皆可不必。

                                                                                              • —凡為夫婦，務宜恩義交至，相敬如賓，至於兒女長大之後，更不宜恃性罵詈，或行箠撻殘恩敗義，莫此之，尤若為婦者欺凌夫主，此則牝鷄司晨，甚為不祥，三綱不正，家道何由興起乎，和氣致祥，戒之勉之。

                                                                                              康熙甲午仲夏
