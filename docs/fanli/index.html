                                                                                              家规-康熙甲午 #

                                                                                              18修，康熙甲午[啓經公]辑，采用：民国庚辰本[1/33]-p12，参考：乾隆丁未本[1/1]-p142。共26条。

                                                                                              • 家规-康熙甲午（1714）

                                                                                              宗约编-嘉庆丁巳辑 #

                                                                                              万历年间，由陈九经整理，并编写《宗约规程》，在嘉庆丁巳年重新整理，原文放在23修。采用：民国庚辰本[1/33]-p047。

                                                                                              • 宗约编

                                                                                              包括：

                                                                                              1. 序文-张元敬
                                                                                              2. 准帖
                                                                                              3. 告示
                                                                                              4. 宗约规程-陈九经
                                                                                              5. 跋-纯仁

                                                                                              創造祠堂碑記-萬曆乙酉(1585) #

                                                                                              萬歷歲次乙酉仲冬月吉日立石。文林郞知浦江縣事，額陵春野兪祖善 俞祖善撰文。署儒學舉人，寧波何汝哲書丹。文林郎知武寕縣事，邑人一亭張元敬 张元敬篆。

                                                                                              原文采用：民国庚辰本[1/33]-p050

                                                                                              见（23修）创造祠堂碑记 创造祠堂碑记

                                                                                              修葺祠堂碑記-崇禎戊辰 #

                                                                                              崇禎戊辰仲冬月吉日。任貴州思南府經歷第二十世祈山派九十七公尚恩 陈尚恩鳩族立石。賜進士文林郎知蘭谿縣事，前浦江縣知縣晉陵莊起元撰文。署浦江縣學舉人閩中朱洧書丹。邑庠增廣生廿一世孫克敏 陈克敏篆額。

                                                                                              原文采用：民国庚辰本[1/33]-p052

                                                                                              见（23修）修葺祠堂碑记 修葺祠堂碑记

                                                                                              甲寅乱后重整宗祠记-同治庚午 #

                                                                                              康熙廿三年歲次甲子小春月望日，第廿二世所山派惠百五十五，元席，薰沐盥手記。原文采用：民国庚辰本[1/33]-p062。

                                                                                              见（16修）甲寅乱后重整宗祠记 甲寅乱后重整宗祠记

                                                                                              同治庚午重建宗祠记-同治庚午 #

                                                                                              同治九年庚午冬月吉旦，辛酉壬戌舉人揀選知縣眷姪樓簡拜。原文采用：民国庚辰本[1/33]-p072。

                                                                                              见（30修）同治庚午重建宗祠记 同治庚午重建宗祠记
