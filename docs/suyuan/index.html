                                                                                              是圖不知其所自來，寫系又殘剥殆盡，無從參考，本不足存，然以刊列旣久，未便以臆見就刪，乃爲詳其大略，訂其舛謬，繪爲一圖，殿於白門石阜之後備之也，亦外之也，學者於是非之際，信者述之，疑者闕焉，況譜牒之作，所以定親疎、决嫌疑、别同異、明是非哉，定其棄取，則俟夫族之有識者。是爲引。
