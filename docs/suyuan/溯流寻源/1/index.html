                                                                                              夏商胡公：授姓始祖

                                                                                              夏商胡公：授姓始祖 #

                                                                                              返回目录

                                                                                              中国历史起于"三皇五帝"。其中"三皇"说法不一，一般指羲皇伏羲、娲皇女娲、黄帝轩辕、炎帝神农。而"五帝"，按照《史纪·五帝本纪》记载为黄帝轩辕、颛顼、帝喾、唐尧、虞舜。

                                                                                              虞帝舜传位给大禹，大禹之子夏启开辟夏朝。虞帝舜后裔"商均"、“虞思"在夏朝被封在虞地，称之为"有虞国”1。

                                                                                              到了商朝，殷汤封舜的后裔虞遂于淮阳陈地。称之为"陈国"2。

                                                                                              周武王灭商朝，仍封舜的后裔于陈国3。其中，虞阏父在周朝任陶正，管理陶器的制作4。而虞阏父之子胡公满，周武王封之于陈国5。

                                                                                              所有陈姓家谱基本都以此开端，奉胡公为受姓始祖。而后面陈国的世系基本都是引用《史记·陈杞世家》中前半部分陈国的记载。

                                                                                              《檀溪陈氏家谱》中胡公有"廣–逵–齊……“一脉，其依据什么文献已无从考证，《溯流尋源繪圖引》也说：“是圖不知其所自來”。

                                                                                              而对于胡公的记载，结合家谱中第九世孙陈应纪在《陈氏源流节要》。成文于宋宝祐丙辰1256年）中做的考证来看，除了前面提到的《左传·襄公二十五年》（文中称为《左氏》）和《史记·陈杞世家》，还引用了《唐书》“夏禹封舜子商均於虞城，三十二世孫遏父爲周陶正，武王妻以元女太姬，生滿，封之陳”6。

                                                                                              世系图人物简传 #

                                                                                              胡公，妫姓，讳满。父仕周，为陶正，武王赖其利器，用与神明之后也，丙戌践祚，妻公以元女太姬，封于太昊之墟，使奉舜祀，是为胡公侯爵，国号陈。后因以为氏。
                                                                                              （“溯流寻源绘图"注：禹舜后裔，周武王封之于陈侯爵，是为得姓受氏之祖。）

                                                                                              申公，讳犀侯。
                                                                                              相公，讳皋羊。继申公袭爵。
                                                                                              孝公，讳突。相公卒，袭。
                                                                                              慎公，讳圉戎。卒于周厉王二十有一年。
                                                                                              幽公，讳宁。立于厉王丁未，卒于己巳，袭爵二十三年。
                                                                                              厘公，讳孝。卒于宣王乙巳。袭爵三十有六年。
                                                                                              武公，讳灵。卒于幽王庚申。
                                                                                              夷公，讳说。立于幽王辛酉，卒于癸亥。
                                                                                              平公，讳燮。夷公弟，嗣立，卒于平王丙戌。
                                                                                              文公，讳圉。卒于平王二十有六年。
                                                                                              桓公，讳鲍。袭爵二十有八年，卒于桓王甲戌。
                                                                                              厉公，讳佗。（依据史记更正为厉公，字敬仲）

                                                                                              注释 #

                                                                                              返回目录

                                                                                              1. 《史记·陈杞世家》“舜已崩，传禹天下，而舜子商均为封国”。唐·司马贞《史记索隐》“商均所封虞，即今之梁國虞城是也”。 ↩︎

                                                                                              2. 汉·宋衷注《世本·陈世》（清·雷学淇校辑本）“陈遂、舜后，厉公跃，共公朔。宋忠曰：虞思之后，箕伯、直柄中衰，殷汤封遂于陈，以为舜后”。 ↩︎

                                                                                              3. 《史记·周本纪》“武王追思先圣王，乃襃封神农之後於焦，黄帝之後於祝，帝尧之後於蓟，帝舜之後於陈，大禹之後於杞”。 ↩︎

                                                                                              4. 《左传·襄公二十五年》“昔虞阏父为周陶正，以服事我先王”。 ↩︎

                                                                                              5. 《左传·襄公二十五年》“庸以元女大姬配胡公，而封诸陈”。 ↩︎

                                                                                              6. 宋·欧阳修、宋祁《新唐书·表第十一·陈氏表》。 ↩︎
