                                                                                              周朝齐国：田氏代齐

                                                                                              周朝齐国：田氏代齐 #

                                                                                              返回目录

                                                                                              恒子以后是釐子田乞，田乞作为齐国大夫，深得民心，后来官至齐国丞相，专权齐国。

                                                                                              一直到太公田和，此时的齐国国君为齐康公，康公沉迷酒色，田和将其软禁到海上，并托魏文候向周天子册封自己为齐候，取代齐康王，并列于各诸侯。从而完成了"田氏代齐"。

                                                                                              到了田因继位，齐国最后一位姜氏1国君齐康公去世，整个齐国真正属于田氏。此时周王室式微，各诸侯纷纷开始称王。田因也开始称王，为齐威王，齐国国力快速发展，成为战国七雄之一。

                                                                                              随着秦国依次灭韩国、赵国、魏国、楚国、燕国，最后，齐国最后一位君主田建2投降秦国，宣布齐国灭亡。田建被秦国流放至共地，并最终死于共地。而秦国也因此完成了统一。田氏四处散轶，有改其他姓氏的，其中有一支仍改回陈姓的。
