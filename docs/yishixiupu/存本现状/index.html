                                                                                              乾隆丁未本（1册） #

                                                                                              这是现在为止能够找到的最老的一个版本，内容刊误最少，可惜仅存1本。其所有内容都将优先采纳整理进本书。
                                                                                              主要内容包括：奕世修谱名衔、公谱编号总目、平粮实绩、谱序、宗谱规条、奕世行第字母图、奕世迁居地里、文武宦绩(文宦/武宦/儒学)、陈氏家乘等。

                                                                                              嘉慶丁丑本（1册） #

                                                                                              嘉庆丁丑本为第26世行传（残缺），从廉928起到廉2034止。

                                                                                              光绪甲辰A本（22册） #

                                                                                              该本世系表和行传较为齐全，刊误较民国庚辰本要少。但是文集作品类流传较少。
                                                                                              第1册主要包括：修谱费用、奕世修谱名衔、公谱编号总目、徵蹟（孝友錄、節賢錄、文武世宦錄等）等。第2至4册为世系表。第5至22册为行传（其中第22册为1至18世行传）。

                                                                                              光绪甲辰B本（3册） #

                                                                                              该本恰好弥补A本中文集作品类较少的不足。其中第1册包括奕世迁居地里、溯流尋源繪圖（硕公之前）、外傳（白門石埠和諸暨白門派）、较为齐全的像赞墓图、奕世行第字母圖（注：比乾隆丁未本详细）、以及公文类（旌表、判案、遺囑、議約、繼約等）。第2册为部分世系图。第3册为1-18世行传，封面显示为"離"字号，应该与A本行传为一套。而A本中归集的第22册（1至18世行传）通过内页判断为洪宪丙辰本。此系电子版归档失误，为便于查找，仍按实际归档分册本。
                                                                                              光绪甲辰本A、B两册合并后，内容较为齐全，刊误也相对较少，不过诗文集和"平糧實蹟"仍然缺失，较为遗憾。本次研究整理过程中除乾隆丁未本外，优先采纳该本。

                                                                                              洪宪丙辰本（1册） #

                                                                                              仅存1册内容基本同光绪甲辰B本的第1册。因此本次研究整理基本未采纳。

                                                                                              民国庚辰本（33册） #

                                                                                              这是目前为止找到最齐全的一个版本，可以通过这个版本窥见《浦陽檀溪陳氏宗譜》全貌。但是可惜的是当时刊印仓促粗糙，内容未能核验校对，错漏较多，导致很多处无法理解阅读，尤其是诗文、公文等，甚至都无法还原写作时的本意，原文采用时将保留这些错漏或不能理解之处。本次研究整理只有在前面乾隆丁未本和光绪甲辰本（A、B）实在缺失的情况下，才会采纳该版本作为研究整理素材。

                                                                                              总结 #

                                                                                              综合来看，乾隆丁未本刊印年份最早、水平最高，因此优先采用。而光绪甲辰A本中22册（1至18世）实际为洪宪丙辰本因此不采用，其余全部采用（乾隆丁未本已有的除外），光绪甲辰B本3册全部采用。而民国庚辰本内容最为齐全，其内容结构可作为本次研究整理的框架（具体见 家谱框架结构分析），至于刊误问题，尽量通过查阅其他相关文献，做到查漏补缺。
